git commands in order:

git init (I didn't use this one)

cd path-or-name-of-a-folder

mkdir name-of-new-folder

cd name-of-new-folder

git clone link-of-github-repository

here we make changes in code etc.
then:

git status

git add .
or
git add name-of-a-file

git status

git commit -m "message" -m "description"

git push origin main
or
git push origin master

git pull
